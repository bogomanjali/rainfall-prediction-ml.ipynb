# 🌧️ Rain Prediction using Machine Learning

This project explores how machine learning can be used to predict whether it will rain based on different weather conditions.

Using historical weather data such as temperature, humidity, wind speed, and other environmental factors, the model learns patterns that help estimate the likelihood of rainfall.

The project was developed using **Python and Google Colab** as part of a machine learning learning exercise.

---

## Project Overview

Weather prediction is an important real-world problem that affects areas like agriculture, transportation, and disaster preparedness.

In this project, we analyze weather data and train a machine learning model that predicts whether rain is likely to occur based on several weather features.

The goal is to understand the relationship between weather variables and rainfall while practicing the full machine learning workflow.

---

## Workflow

The notebook follows a typical machine learning pipeline:

• Loading and exploring the weather dataset
• Data cleaning and preprocessing
• Visualizing patterns in the data
• Training a machine learning model
• Evaluating the model’s performance
• Making rainfall predictions based on weather conditions
